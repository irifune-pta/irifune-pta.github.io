# 入船小学校 PTA安全活動 ホームページ

PTA安全活動の案内・予約・実績公開のためのウェブサイトです。

公開URL: https://irifune-pta.github.io/

---

## 📁 ファイル構成

```
/
├── index.html        ← メインページ（基本的に触らない）
├── config.js         ← 年度・URL設定（年度更新時に編集）
├── schedule.js       ← クラス担当期間データ（年度更新時に編集）
├── README.md         ← このファイル（運営マニュアル）
├── robots.txt        ← 検索エンジン除外設定
├── images/
│   ├── school-emblem.png            ← 校章
│   ├── 安全活動年間予定表v2.png     ← 年間予定表
│   ├── A.登校時安全指導v2.png       ← 朝の見守り場所マップ
│   ├── B.放課後安全パトロールv2.png ← 放課後パトロールマップ
│   └── (その他の写真)
└── PTA安全活動_予約シート_テンプレート.xlsx  ← 予約管理の予備テンプレート
```

## 📝 年度更新の手順（毎年やること）

### Step 1: `config.js` を編集

テキストエディタ（メモ帳など）で `config.js` を開き、以下の項目を新年度の値に書き換えます。

| 項目 | 説明 | 例 |
|------|------|-----|
| `schoolYear` | 年度の表示文字列 | `"令和9年度"` |
| `siteUrl` | このホームページのURL | `"https://irifune-pta.github.io/"` |
| `formUrl` | 予約・報告フォームの公開URL | `"https://forms.gle/XXXXXXXX"` |
| `reservationStatusUrl` | 予約状況スプレッドシートの埋め込みURL | （後述） |
| `reservationStatusViewUrl` | 予約状況スプレッドシートの全画面URL | （後述） |
| `reportChartUrl` | 活動実績グラフの埋め込みURL | （後述） |
| `reportChartViewUrl` | 活動実績グラフの全画面URL | （後述） |

**スプレッドシートのURL取得方法：**

1. 該当のスプレッドシートを開く
2. メニューの「ファイル」→「共有」→「ウェブに公開」
3. 「埋め込む」を選択して、公開
4. 表示されるiframeコードの `src="..."` 部分のURLが「埋め込みURL」
5. そのURLからクエリパラメータ（`&single=true&widget=true&headers=false`）を除いたものが「全画面URL」

### Step 2: `schedule.js` を編集

クラス別担当期間データを新年度の日付に書き換えます。

```javascript
"6年1組": { periods: [["2026-07-01", "2026-07-15"]] },
            ↑開始日       ↑終了日（YYYY-MM-DD形式）
```

期間が複数に分かれる場合（夏休みを挟むなど）：

```javascript
"6年2組": {
  periods: [
    ["2026-07-16", "2026-07-21"],
    ["2026-09-01", "2026-09-15"]
  ],
  note: "夏季休業期間を除く"  ← 補足を表示
},
```

### Step 3: `index.html` の「クラス別 担当期間」表示を更新

`index.html` を開いて、各 `<div class="grade-card">` 内の日付表示を新年度に書き換えます。

**`schedule.js` だけ更新しても、画面表示は更新されません。** カレンダー登録機能は `schedule.js` を読みますが、画面の日付表示は `index.html` 内のテキストを使っています。

該当箇所（17クラス分）：
```html
<div class="class-row" data-cls="6年1組">
  <span class="class-name">6年1組</span>
  <div class="class-period">7月1日(水) 〜 7月15日(水)</div>  ← この日付を編集
</div>
```

### Step 4: 画像の差し替え（必要があれば）

通学路・パトロールマップやスケジュール表に変更があれば `images/` 内のファイルを同名で上書き保存します。

| ファイル | 内容 |
|----------|------|
| `安全活動年間予定表v2.png` | 年間予定表 |
| `A.登校時安全指導v2.png` | 朝の見守り場所マップ |
| `B.放課後安全パトロールv2.png` | 放課後パトロールマップ |

ファイル名を変えると `index.html` の参照も更新が必要になります。**同名で上書き**するのが楽です。

### Step 5: GitHubにアップロード

1. GitHubリポジトリ（https://github.com/irifune-pta/irifune-pta.github.io 等）を開く
2. 編集したファイルをドラッグ&ドロップ、または「Upload files」ボタンで上書き
3. 下の「Commit changes」ボタンで保存

通常1〜2分で公開URL（https://irifune-pta.github.io/）に反映されます。

## 🔧 その他の更新方法

### 緊急連絡先の電話番号を変更したい

`index.html` 内の `tel:` リンクを直接編集します。

```html
<div class="num"><a href="tel:047-353-8503">047-353-8503</a></div>
```

### Googleフォームの内容を変更したい

Googleフォーム自体を編集してください。**フォームURLが変わらない限り、サイト側の変更は不要**です。

新しいフォームを作り直した場合は、`config.js` の `formUrl` を更新します。

### スプレッドシートを差し替えたい

新しいスプレッドシートを「ウェブに公開」設定にして、得られたURLで `config.js` の以下を更新します：
- `reservationStatusUrl` / `reservationStatusViewUrl`
- `reportChartUrl` / `reportChartViewUrl`

### 校章やマップを変更したい

`images/` フォルダのファイルを同名で上書きします。

校章 (`school-emblem.png`) は `index.html` 内に **base64で埋め込まれている**ため、変更時は再エンコードが必要です。技術的に難しい場合は、エンジニアに相談してください。

### お知らせ文（「分担表の廃止について」）を変更したい

`index.html` 内の以下を編集します：

```html
<section id="notice" class="notice-section">
  ...
</section>
```

## 🚀 デプロイ（公開）の仕組み

このサイトは **GitHub Pages** で公開されています。

- リポジトリにファイルをコミット → 自動的に公開URLに反映
- 別のサーバーやサービスは不要
- コストはかかりません

**注意：** リポジトリは `public`（公開）設定です。中身は誰でも閲覧できます。
氏名・電話番号などの個人情報は、フォーム経由でGoogleに送信される設計のため、リポジトリ自体には個人情報を含めないでください。

## 🛠 トラブルシューティング

| 症状 | 対処 |
|------|------|
| **画像が表示されない** | ファイル名（特に `v2` の有無や全角・半角）を確認 |
| **文字化けする** | ファイルがUTF-8で保存されているか確認 |
| **カレンダー登録が動かない** | `schedule.js` の構文エラー（カンマ・括弧）を確認 |
| **URLが古いまま表示される** | ブラウザの強制再読み込み（Ctrl+F5）でキャッシュをクリア |
| **iframeが表示されない** | スプレッドシートの「ウェブに公開」設定が有効か確認 |
| **GitHubにコミットしたが反映されない** | 1〜5分待つ／GitHub Actionsのログを確認 |

## 📚 用語

- **年度**：4月〜翌年3月の1年間。例：「令和8年度」は 2026年4月〜2027年3月
- **登校時安全指導 (A)**：朝の通学路で旗振りなど見守り活動
- **放課後安全パトロール (B)**：放課後の街区を巡回する活動
- **iframe**：ウェブページに別サイトを埋め込む仕組み（スプレッドシート表示などに使用）
- **base64**：画像などをテキストに変換する形式。HTMLに直接画像を埋め込める
